## Desafio Técnico Veeries 

O desafio consiste em criar um sistema para acompanhar diariamente a previsão de dados meteorológicos do Centro-Oeste do Brasil.
Especificações: 

1) Obter dados de previsão em resolução diária (chuva, temperaturas, umidade) 

2) Sistematizar um processo de atualização diária 

3) Local: Centro Oeste do Brasil (por estado, preferencialmente por município)
